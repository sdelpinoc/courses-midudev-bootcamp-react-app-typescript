# React + Typescript
  - To start a project, run: 
    - `yarn create vite`
   - Set name or type .(dot), select React and Typescript

  - Then run:
    - `yarn`

  - To run the project:
    - `yarn dev`

  - To run the server:
    - `yarn run json-server`

## Installations
  - Json server:
    - `yarn add --dev json-server`

  - Axios:
    - `yarn add axios`

## Info
  - File with .d in the name are only for definitions, ex:
    - types.d.ts
